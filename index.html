Solución

Alcance de la Solución Técnica

Arquitectura de vigilancia automatizada diseñada para capturar, procesar y alertar sobre cambios críticos en el entorno normativo colombiano.

Fuentes de Scraping

  • • Normogramas del Estado (SUIP/Sectoriales)
  • • Gacetas Oficiales y Diario Oficial
  • • Alertas de Google Noticias (Vigilancia de mercado)

Motor de Inteligencia

  • • Match Temático mediante implementación de IA
  • • Alertas automáticas vía correo electrónico
  • • Dashboard administrativo centralizado

Tiempos

Plan de Trabajo Detallado

01 MES 1: Ingeniería de Datos e Implementación IA

Semana 1: Diagnóstico de Fuentes

Mapeo y Arquitectura de Datos

  • Identificación de endpoints técnicos en SUIP y Diario Oficial.
  • Definición de estructura de base de datos para almacenamiento de normas.
  • Configuración de entorno de nube (Servidores y BD).

Semana 2: Desarrollo de Scrapers

Extracción Automatizada Core

  • Construcción de bots de scraping para Normogramas y Gacetas.
  • Implementación de sistemas de limpieza de texto (Parsing de PDF a texto plano).
  • Protocolos de re-intento ante caídas de sitios gubernamentales.

Semana 3: Orquestación de IA

Implementación de Lógica Cognitiva

  • Configuración de APIs de IA para procesamiento de lenguaje natural.
  • Desarrollo del algoritmo de "Match Temático" basado en perfiles de cliente.
  • Integración con Google News API para monitoreo de prensa.

Semana 4: Pruebas de Ingesta

Validación de Datos Base

  • Ajuste de precisión en el emparejamiento de normas por etiquetas.
  • Optimización de velocidad de lectura de archivos oficiales.
  • Primer reporte de prueba generado internamente.

02 MES 2: Plataforma Administrativa y Go-Live

Semana 5: Desarrollo de Interfaz

Consola de Administración (Backend)

  • Panel para gestión de clientes y palabras clave (Keywords).
  • Visualizador de historial de alertas emitidas.
  • Configuración de usuarios administradores de Niubox.

Semana 6: Sistema de Notificaciones

Motor de Correo y Reportes

  • Diseño de plantillas de correo automatizadas.
  • Configuración de frecuencia de alertas (Inmediatas vs. Semanales).
  • Integración con servicio de despacho de emails masivos.

Semana 7: Control de Calidad

User Acceptance Testing (UAT)

  • Periodo de pruebas con el equipo legal de Niubox.
  • Ajuste de filtros de IA para reducir "ruido" en las alertas.
  • Refinamiento de experiencia de usuario en el Dashboard.

Semana 8: Despliegue Final

Lanzamiento a Producción

  • Migración a entorno final de producción.
  • Capacitación del equipo administrativo.
  • Activación de monitoreo 24/7 y soporte de operación.
